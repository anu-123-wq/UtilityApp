ABOUT THE PROJECT

Web Utility App is a PWA(Progressive Web App) which has some major advantages over both web and native app features with the features like:

-> Linkability: One of the most powerful features of the web is the ability to link to an app at a specific URL without the need for an app store or complex installation process.

-> Network Independent: It can be worked where the network is very unreliable or where the connectivity is very poor.

-> Responsiveness: It's UI will fit in any form factor be it desktop, mobile, tablet or whatever come next.

-> Secure: The connection between the user and App are secured against any third parties.
